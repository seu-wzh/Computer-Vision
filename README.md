# 计算机视觉

计算机视觉是人工智能领域的一个重要分支，旨在使计算机系统能够像人类视觉系统一样理解和解释图像或视频数据，主要目标是通过模拟人类视觉系统的方式，使计算机能够识别、理解和解释数字图像或视频中的内容。本项目为东南大学 2023-2024 学年中人工智能专业课程《计算机视觉》的课程笔记、实验以及课程项目。
